MULTITHER 2.0 v51 - Firebase Setup Summary
==========================================

This game uses Firebase Realtime Database and Firebase Authentication.

Files that must be uploaded to GitHub Pages:
- index.html
- style.css
- game.js
- firebase-config.js
- background.jpg
- gameLogo.png
- SlitheringNumbers.mp3
- ICON.ico

Firebase paths used:
- multither2_scoreboard
  Public live leaderboard. Pupils can see ranking.

- multither2_score_records
  Protected online records for teacher/admin CSV export.

- roles/admins
  Admin UID list.

- roles/teachers
  Teacher UID and assigned school access.

Please read ADMIN_TEACHER_EXPORT_SETUP.txt for the full setup and recommended database rules.

MULTITHER 2.0: Multiply and Play!
Educational upgrade notes

Main upgrades included:
1. Bilingual English and Simplified Chinese interface.
2. Practice Mode for supported learning and Challenge Mode for fluency evidence.
3. Keyboard controls: Arrow Keys or WASD. Mobile-friendly on-screen arrow controls are also included.
4. Answer apples and a more child-friendly snake visual.
5. Learning tracker showing attempts, correct answers, accuracy, and streak.
6. End screen separates Game Score from Learning Performance.
7. Fluency evidence includes accuracy, speed, average response time, and fluency rate.
8. Three-star rating and mastery feedback.
9. CSV export for teacher record keeping.
10. Cleaner code structure for GitHub deployment.
11. CSV export now accumulates all saved play attempts from the same browser/device into one file.

Suggested classroom use:
- Practice Mode: use during guided or independent practice.
- Challenge Mode: use after practice to collect fluency evidence.
- Let each pupil play, return to menu, enter the next pupil's name, and continue.
- At the end of the session, click Export CSV once to download one cumulative results file.

CSV Chinese Display Note:
The CSV export includes a UTF-8 BOM so Microsoft Excel can read Simplified Chinese labels correctly instead of showing garbled text.

Important note about accumulated CSV:
The accumulated records are saved in the browser's localStorage. This means the CSV will accumulate results only on the same device and same browser. If pupils play on different computers, each computer will generate its own accumulated CSV file.

CSV accumulation debug update:
- The game now uses a safer combined history system: memory + sessionStorage + localStorage.
- Each completed play attempt is saved automatically at the Game Over screen.
- The end screen now shows how many attempts are saved in this browser.
- Export CSV downloads all saved attempts from this browser/device.
- The game cannot directly append rows into an already-downloaded CSV file because browsers block websites from editing local files automatically. Export once at the end of the class session for one combined file.
12. Movement controls are smoother on keyboards and touch screens: Arrow Keys/WASD support, pointerdown response, no duplicate touch/click input, and quick-turn direction buffering.

Speed adjustment note:
- Snake speed has been downgraded for younger pupils: Slow = 720 ms, Moderate = 520 ms, Fast = 360 ms per movement step.

Latest practice-mode update:
- Practice Mode now stops automatically after 30 questions.
- Pupils can choose End Game only after completing at least 10 questions.
- If they try to stop before 10 questions, the game reminds them to continue first.

Speed levels included: Slow (720 ms), Moderate (520 ms), Fast (360 ms), and Very Fast (260 ms).
